musical_adventures
    by Tyler Montgomery
    http://ubermajestix.com

== DESCRIPTION:

gem to help submit content to ubermajestix.com

== FEATURES/PROBLEMS:
Submit 
lyrics
-and-
adventures
-!-
== SYNOPSIS:
lyric "lyrical text goes here"
adventure --album boom --artist boomers --track boom part 1 --playlist Things that go Boom

== REQUIREMENTS:
none just use it

== INSTALL:

sudo gem install ubermajestix-musical_adventures
